# Best Feature From Each Competitor — Cost / Value / Product-Benefit

> "Steal the best from everyone." For each competitor: their **single standout feature**, why it wins for
> them, the **product benefit** if we adopt it, the **build cost**, and a **verdict** (Adopt / Adapt / Skip).
> Companion to `competitive-analysis.md` and `tier0-roadmap.md`.
>
> Cost = T-shirt size incl. whether we already have foundations: **S** (days–2wk), **M** (3–6wk),
> **L** (1–2 quarters), **XL** (multi-quarter / org-level).
> Value lens = which growth lever it pulls: **Acquisition · Conversion · Retention · Revenue · Clinical · Trust**.
>
> ⚠️ Competitor features are market knowledge (≤2026) — directionally accurate, **verify before external use**.

Created 2026-05-31.

## F. The combination nobody else has (our winning hand)

Each competitor is strong on **one** axis. If we steal the best of each onto our existing
provider-tooling + financial/compliance + 3-app foundation, the combined product is unique:

> **Wysa's AI companion + Spring's triage + Lyra's outcomes + BetterHelp's onboarding + SimplePractice's
> clinician tools + Amaha's care continuum + YourDOST's EAP — on top of payouts/RBI-invoicing/ledger +
> DPDP security + native patient & clinician apps.** No single competitor spans this. That breadth,
> executed well, is the tier-0 position.
