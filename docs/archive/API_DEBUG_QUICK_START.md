## ğŸ¯ Common Use Cases

### Debug Login Issues
1. Click login
2. Open debug panel
3. Find `POST /auth/login`
4. Check if response has `accessToken`

### Debug 401 Errors
1. Filter to "Errors"
2. Look for red requests
3. Check if `Authorization` header exists
4. Verify Bearer token format

### Copy Request for Bug Report
1. Expand the failed request
2. Click **[Copy]** on Request Body
3. Click **[Copy]** on Response Body
4. Paste into your bug report
